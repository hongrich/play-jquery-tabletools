h1. jQuery DataTables TableTools Play! Module

This module makes it easy to include jQuery DataTables TableTools resources in your Play! project.

To use TableTools on your (existing) jQuery datatable:

# Include this module by adding the following line to application.conf:
<pre><code>module.jquery-tabletools=${play.path}/modules/jquery-tabletools</code></pre>

# Add routes to your conf/routes file:
<pre><code>GET     ${ctx}/jquery-tabletools                      module:jquery-tabletools</code></pre>

# Include this script tag in your page:
<pre><code><script src="@{'/public/js/TableTools.min.js'}" type="text/javascript" charset="utf-8"></script></code></pre>

# Include this style in your page:
<pre><code><link rel="stylesheet" type="text/css" href="@{'/public/css/TableTools.css'}">
<link rel="stylesheet" type="text/css" href="@{'/public/css/TableTools_JUI.css'}"></code></pre>

# Include the TableTools using 'T' in the "'sDom' setting":http://www.datatables.net/usage/options#sDom, and set the path to the swf.
<pre><code>  $('#datatable').dataTable({
    ...
    "sDom": '<"H"lT>rt<"F"ip>',
    "oTableTools": {
      "sSwfPath": "@{'/public/swf/copy_cvs_xls_pdf.swf'}"
    },
  });</code></pre>
